Linux常用用户与用户组管理命令（三）

2020-09-03

一、用户组管理命令

用户组管理命令 作用
cat /etc/group 查看用户组
groupadd test 新建名为 test 用户组
groupmod -n test1 test 把名为 test 用户组改为 test1
groupdel test1 删除名为 test1 的用户组

二、用户的管理命令

用户的管理命令 作用
cat /etc/passwd 查看用户
useradd user1 -c test 增加用户 user1 并备注注释信息
useradd user1 -g test 增加用户 user1 到名为 test 的用户组中
usermod -l user2 user1 把名为user1的用户改为 user2
usermod -g test1 user1 将 test2 用户移动至新组 test1 中
usermod user1 -c 注释内容 修改用户注释信息
userdel user1 删除用户user1

三、passwd命令

密码设置命令 左右
passwd 为 root 用户设置密码
passwd user1 为名为 user1 的用户设置密码
