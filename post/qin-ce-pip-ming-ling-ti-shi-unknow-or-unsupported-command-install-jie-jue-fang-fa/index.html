亲测：pip命令提示unknow or unsupported command install解决方法

2020-09-03

pip命令提示unknow or unsupported command install

最近在学习Appium，运行 cmd 使用 pip install Appium-Python-Client命令
出现了下面图片所示的报错信息：

解决办法：

把pip命令稍加变动【pip → pip.exe 或者 pip3】：

C:\Users\86176>pip.exe install Appium-Python-Client
C:\Users\86176>pip3 install Appium-Python-Client

操作执行成功：

至此，pip操作命令执行成功，希望可以帮到你们！
