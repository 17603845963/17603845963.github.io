Linux中已部署项目操作命令

2020-09-04

启动云商：
/opt/lampp/lampp start

关闭防火墙
systemctl stop firewalld

启动Docker
systemctl start docker

设置开机自启 Docker
systemctl enable docker

执行如下命令进入容器内部
docker exec -it jmeterdb bash

执行influx命令进入命令台
influx

数据库时间同步
ntpdate pool.ntp.org

查看当前MeterSphere当前状态
msctl status

服务相关命令
msctl --help

status 查看 MeterSphere 服务运行状态

start 启动 MeterSphere 服务

stop 停止 MeterSphere 服务

restart 重启 MeterSphere 服务

reload 重新加载 MeterSphere 服务

uninstall -v 卸载 MeterSphere 服务

version 查看 MeterSphere 版本信息
