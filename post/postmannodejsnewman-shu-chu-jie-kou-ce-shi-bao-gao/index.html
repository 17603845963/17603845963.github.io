Postman+Nodejs+Newman输出接口测试报告

2020-09-03

1、先安装node.js

官网下载地址：https://nodejs.org/en/download/

附上我的百度云网盘链接：
链接：https://pan.baidu.com/s/1lVzY2ZNbCbliLoFR8VHbXA 提取码：w3lk

【下载完成后就傻瓜式安装，建议默认C盘就行】

2、Win+R，cmd打开命令行，安装cnpm

npm install -g cnpm --registry=https://registry.npm.taobao.org

3、安装newman

cnpm install newman --global

4、检查是否安装成功

newman -v


出现以上版本号提示，表示安装成功；

5、安装reporter

cnpm install -g newman-reporter-html

6、使用newman命令行执行测试

- 导出集合为json脚本

- 导出环境为json文件

- 执行命令

newman run 脚本 -e 环境文件 -n 循环次数 -r html --reporter-html-export 报告名称
